## 1. 产品简介

### 1.1 产品概述

Q版方块人大乱斗是一款融合中国武术、西方魔法、东方仙术、黑巫术四大战斗体系的多人实时格斗竞技游戏。游戏采用Q版方块人角色，包含12个来自不同文明的英雄，支持1V1决斗和5V5团战等丰富游戏模式。游戏免费游玩，所有影响数值的道具完全免费，付费内容仅为外观。

### 1.2 核心功能一览
| 功能模块 | 功能说明 |
|---------|---------|
| 1V1决斗 | 经典单挑模式，BO5三局两胜制 |
| 5V5团战 | 10人团队对战，含野怪/Boss/资源点争夺 |
| 训练场 | 木人桩/武器架/Boss模拟/对战擂台自由练习 |
| 闯关挑战 | 100波PVE关卡，有全球排行榜 |

| Battle Pass | 赛季通行证，免费线+付费线 |

## 2. 游戏安装

### 2.1 网页版
1. 打开浏览器（推荐Chrome）
2. 访问游戏URL：`https://play.qbf.game`（上线后地址）
3. 等待加载完成
4. 输入昵称，选择角色，开始游戏

### 2.2 Steam版
1. 打开Steam客户端
2. 搜索"Q版方块人大乱斗"
3. 点击"免费游玩"添加到库
4. 等待下载安装完成
5. 启动游戏

### 2.3 首次使用引导
1. 启动游戏后进入主界面
2. 完成新手教程（推荐首次游玩必玩）
3. 进入训练场熟悉操作
4. 尝试1V1匹配对战
5. 体验更多游戏模式

## 3. 基本操作

### 3.1 操作键位

| 操作 | 按键 | 说明 |
|------|------|------|
| 移动 | W/A/S/D | 前/左/后/右移动 |
| 视角控制 | 鼠标移动 | 自由视角旋转 |
| 轻攻击 | 鼠标左键 | 速度快，可连按形成连招 |
| 重攻击 | 鼠标右键 | 伤害高，有前摇，可蓄力 |
| 闪避/翻滚 | Shift | 消耗体力，有无敌帧，方向可控制 |
| 格挡/弹反 | 空格 | 消耗体力减免伤害，精确时机触发弹反 |
| 技能 Q | Q | 角色专属技能1 |
| 技能 E | E | 角色专属技能2 |
| 技能 R | R | 角色大招（终极技能） |
| 切换武器 | 1 / 2 | 切换主武器/副武器 |
| 使用道具 | F | 使用拾取的回血道具等 |
| 聊天/表情 | Enter / T | 打开聊天框 / 快捷表情 |

### 3.2 战斗系统说明

#### 连招系统
- **轻攻击连段**：最多4连击（轻→轻→轻→轻），每段伤害递增
- **重攻击插入**：在轻攻击连段中可插入重攻击，触发不同终结技
- **挑飞追击**：部分连招可将敌人挑飞，空中可追击

#### 体力系统
- 攻击、闪避、格挡均消耗体力
- 体力耗尽时角色进入虚弱状态（无法闪避和格挡）
- 体力随时间自动恢复

#### 弹反（完美格挡）
- 在对手攻击命中的精确时机按下格挡，触发弹反
- 弹反成功可弹开对手，造成硬直并触发反击窗口
- 弹反是高端对战的核心技巧

#### 霸体机制
- 部分重攻击/大招自带霸体，不被打断
- 霸体状态仍然会受到伤害

#### 击杀回复
- 击杀玩家后立即恢复50%最大生命值
- 鼓励连续战斗，击杀奖励积极反馈

#### 脱战恢复
- 脱离战斗5秒后开始自动回血
- 坦克/辅助回血最快（每秒3%），刺客最慢（每秒1%）

### 4.2 力量体系克制关系
```
武术 → 克制 → 魔法（近身打断施法）
魔法 → 克制 → 巫术（远程净化驱散）
巫术 → 克制 → 仙术（诅咒削弱神圣）
仙术 → 克制 → 武术（护盾抗住近战）
```

### 4.3 双武器切换
- 战斗中按1/2随时切换武器，无冷却
- 切换瞬间有0.3秒切换动画，期间可被打断
- 切换后第一击伤害+20%（切换加成）
- 两把武器共享技能冷却，但部分技能效果随武器切换变化

### 4.4 双美术风格
- **Q版风格**：二头身/三头身，大头大眼，卡通特效（默认）
- **朋克版风格**：机械义肢、霓虹光效、能量武器（商城/BP解锁）
- 两种风格仅改变外观，技能机制和数值完全一致

## 5. 游戏模式

### 5.1 1V1 决斗
- **规则**：BO5五局三胜制
- **地图**：龙城演武、罗马竞技场、竹林幽径（3张）
- **胜利条件**：击杀对手3次获胜
- **特色**：支持观战系统，其他玩家可实时围观

### 5.2 5V5 团战
- **规则**：5人组队 vs 5人
- **地图**：北海冰原、长安城、波斯花园（2张起步）
- **目标**：击杀对方全员 / 摧毁据点 / 积分制
- **地图元素**：
  - 野区野怪（刷装备和经验）
  - 资源点（争夺控制）
  - Boss刷新点（击杀获得陨铁武器）
  - 可破坏环境
  - 高低差地形

### 5.3 训练场
训练场是无门槛的开放世界式空间，所有玩家随时可进入。

| 区域 | 功能 |
|------|------|
| 木人桩区 | 自由练习连招，显示伤害数值和连招评分 |
| 武器架区 | 试用所有武器类型，熟悉攻击模组 |
| 闯关挑战区 | 单人AI关卡，逐层递进，有全球排名 |
| Boss模拟区 | 模拟Boss战，练习团队配合 |
| 对战擂台 | 玩家之间自由切磋，无排名无惩罚 |

### 5.4 闯关挑战
- 100波PVE关卡，由弱到强递进（普通→精英→Boss）
- 每关有时间限制和评分
- 评分维度：通关时间、连招连击数、无伤次数、所用武器多样性
- 全球排行榜：每个关卡、每个角色、每把武器都有独立排名
- 通关奖励：武器箱、金币、角色碎片、专属称号

## 10. 常见问题（FAQ）

### 连接相关

**Q：游戏连接失败怎么办？**
A：1. 检查网络连接是否正常 2. 确认服务器是否在维护（查看公告）3. 尝试刷新页面/重启游戏 4. 检查防火墙是否阻止了游戏连接

**Q：游戏中延迟很高怎么办？**
A：1. 使用有线网络代替WiFi 2. 关闭后台下载/视频等占用带宽的程序 3. 尝试更换DNS（如8.8.8.8）4. 联系网络运营商检查网络质量

**Q：游戏掉线了怎么办？**
A：1. 重新连接游戏（5分钟内重连保留对局）2. 检查网络稳定性 3. 如频繁掉线请联系客服

### 操作相关

**Q：操作不响应/卡顿怎么办？**
A：1. 检查帧率是否正常（设置中显示FPS）2. 降低画质设置 3. 关闭其他占用资源的程序 4. 更新显卡驱动

**Q：如何提升格斗技巧？**
A：1. 先在训练场木人桩练习连招 2. 理解弹反时机（精确格挡）3. 掌握体力管理 4. 学习体系克制关系 5. 多看高手对局回放

**Q：武器切换后第一击没有伤害加成？**
A：切换加成仅在切换动画完成（0.3秒）后的第一击生效，切换过程中被打断则不生效。

### 角色相关

**Q：如何解锁新角色？**
A：所有12个角色免费使用，无需解锁。

**Q：Q版和朋克版风格有什么区别？**
A：仅视觉外观不同，技能机制和数值完全一致。Q版为默认风格，朋克版可通过商城购买或Battle Pass获取。

**Q：力量体系克制是什么意思？**
A：武术克魔法、魔法克巫术、巫术克仙术、仙术克武术。被克制方在对战中处于一定劣势，需要通过操作弥补。

### 账号相关

**Q：游戏进度会保存吗？**
A：是的，排位段位、对局记录、商城购买等数据都会保存在服务器上。

**Q：支持跨平台游玩吗？**
A：目前仅支持PC端，移动端和主机端将在后续版本适配。
