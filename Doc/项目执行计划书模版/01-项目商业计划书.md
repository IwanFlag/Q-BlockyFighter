## 1. 执行摘要

### 1.1 项目概述
Q版方块人大乱斗（Blocky Fighter）是一款多人实时动作竞技游戏，融合流星蝴蝶剑经典格斗手感、永劫无间美术风格与 LOL 的 UI 角色设计理念。游戏采用 **Q版 + 朋克版** 双风格体系，首发 12 位来自不同文明的英雄角色，覆盖武术、巫术、魔法、仙术四大力量体系，每位角色配备双专属武器与独立连招系统。

游戏核心玩法为 **无锁定格斗 + 帧同步多人对战**，支持 1V1 决斗和 5V5 团队对战模式，配合训练场闯关系统，提供丰富的 PvE 与 PvP 体验。商业模式采用 **纯外观付费 + Battle Pass 赛季制**，确保公平竞技环境。

目标首发平台为 **Steam（PC）+ Web**，后续拓展至移动端和主机平台。

### 1.2 愿景与使命
- **愿景：** 打造全球最具特色的多人格斗竞技游戏，让来自不同时空的武者在同一个战场上书写传奇。
- **使命：** 以极致的格斗手感和丰富的角色体系，为玩家提供既硬核又友好的竞技体验。

### 1.3 核心亮点
| 维度 | 说明 |
|------|------|
| 解决的问题 | 市场缺乏兼具流星蝴蝶剑硬核格斗手感与 LOL 英雄深度的多人竞技游戏 |
| 解决方案 | 12角色×4力量体系×12武器×双武器切换×连招系统，构建深度格斗体验 |
| 市场机会 | 全球格斗/竞技游戏市场持续增长，免费游玩+内购模式降低门槛 |
| 竞争优势 | 独创四大力量体系克制 + Q版/朋克版双风格 + 帧同步格斗 + 纯外观付费公平竞技 |

### 1.4 为什么是现在（Why Now）
| 趋势 | 现状 | 我们的机会 |
|------|------|-----------|
| 免费竞技游戏崛起 | 《永劫无间》转免费后玩家激增，证明 F2P+外观付费在格斗品类可行 | 借鉴成功模式，切入细分赛道 |
| Web 游戏技术成熟 | Three.js/WebGPU 让浏览器 3D 游戏品质大幅提升 | Web 原型零门槛获客，快速验证 |
| Unity 跨平台生态成熟 | Unity 对 PC/Web/移动端覆盖完善 | 一次开发多端部署，降低移植成本 |
| 短视频传播驱动 | 抖音/B站/Twitch 游戏内容爆发，格斗游戏观赏性强 | Q版风格天然适合短视频传播 |

> 💡 **提示：** "Why Now"是投资人判断时机是否合适的关键。永劫无间证明格斗品类 F2P 可行，Web 技术降低了原型验证成本，短视频生态为游戏传播提供了天然土壤——三个窗口同时打开，正是入场时机。

## 4. 产品与服务

### 4.1 产品概述
Q版方块人大乱斗是一款以「裂隙纪元」为世界观的多人实时动作竞技游戏。玩家操控来自不同时空文明的武者，使用各自的专属武器和连招系统进行格斗对战。

### 4.2 核心功能
1. **无锁定格斗系统：** 自由视角+手动瞄准，轻/重攻击组合+连招系统+闪避格挡弹反+体力管理
2. **12 角色×4 力量体系：** 武术/巫术/魔法/仙术四体系相互克制，每位角色双专属武器+独立连招
3. **双武器切换：** 战斗中随时切换两把专属武器，切换后首击+20%伤害加成
4. **多模式对战：** 训练场（木人桩/闯关/Boss 模拟/自由切磋）+ 1V1 排位 + 5V5 团战
5. **赛季系统：** 七段位排位 + Battle Pass 赛季制 + 纯外观商城

### 4.3 技术架构
- **客户端：** Unity 3D（C#）/ Web 端 Three.js
- **服务器：** .NET8 帧同步服务器 + WebSocket 通信
- **架构模式：** 帧同步（Deterministic Lockstep），所有模式统一采用
- **最大同屏：** 10人（5V5）
- **目标帧率：** PC 60fps
- **代码仓库：** github.com/IwanFlag/Q-BlockyFighter

|---------|---------|---------|--------|

| DLC 角色包/地图包（可选） | ¥18-38/包 | 15% | 85%+ |

### 5.2 客户获取策略
- **获客渠道：** Steam 商店自然流量、B站/抖音短视频传播、游戏主播试玩、Discord 社区运营、Reddit/贴吧口碑传播
- **获客成本（CAC）：** 目标 < ¥5/用户（F2P 模式下自然流量为主）
- **客户生命周期价值（LTV）：** 目标 ¥30-50/用户（12 个月周期）
- **LTV/CAC 比率：** 目标 > 6

### 5.3 客户获取漏斗
```
潜在用户总量：500万（Steam 格斗品类用户池）
    ↓ 2% 转化率
下载/注册：10万
    ↓ 30% 转化率
活跃用户（月活）：3万
    ↓ 10% 转化率
付费用户：3000
    ↓ 60% 续费率
长期付费客户：1800
```

### 5.4 增长飞轮
```
免费游玩 → 大量玩家涌入 → 优质对局体验 → 短视频/直播传播 → 更多玩家涌入
                                                            ↓
赛季更新/BP奖励 → 活跃度提升 → 付费转化 → 收入增长 → 更多内容投入 → 飞轮加速
```

## 10. 风险分析与应对

| 风险类型 | 风险描述 | 发生概率 | 影响程度 | 应对措施 |
|---------|---------|---------|---------|---------|

## 11. 竞争壁垒

### 11.1 当前壁垒
- 独创四大力量体系克制系统，差异化角色设计
- Q版+朋克版双风格体系，覆盖休闲与硬核用户
- 帧同步格斗+连招系统，技术门槛较高
- 开源仓库建立技术社区信任

### 11.2 壁垒构建路径
```
Year 1: 产品差异化（独特角色体系+格斗手感）         🟡 弱
Year 2: 社区生态（活跃玩家社区+内容创作者）          🟡 中
Year 3: 赛季内容积累（角色/地图/皮肤库）             🟢 强
Year 4+: 品牌认知+电竞生态                           🟢 极强
```

### 11.3 不可复制性分析
格斗手感需要大量打磨迭代，角色体系设计需要对格斗品类的深度理解。Q版+朋克版双风格的美术资产投入也是后来者的成本门槛。更重要的是，社区一旦建立，玩家的社交关系和内容沉淀形成了自然的迁移成本。
